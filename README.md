# \# FitTrackDB — Individual Project 2026

# 

# The final project is an individual software application developed using MySQL, Python (Flask), and Java. The goal of the project is to design and implement a health and fitness tracking system that stores and manages user data in a structured relational database.

# 

# The system focuses on organizing fitness-related data that is typically spread across multiple platforms. Instead of using separate apps for meals, workouts, and sleep tracking, this project brings all data into one database, allowing for easier management and querying.

# 

# The project includes a MySQL database designed using normalization principles up to Third Normal Form (3NF). The database enforces relationships through foreign keys and includes constraints such as CHECK conditions and cascading deletes to maintain data integrity.

# 

# The application provides functionality for user account creation, login, and profile management. Users can log meals with nutritional information, track workouts and exercises, record body measurements, monitor sleep and energy levels, and log daily water intake. The system also supports goal setting and progress tracking.

# 

# The backend is implemented using Python and Flask to handle routing and database communication. A simple frontend is built using HTML and CSS for user interaction. In addition to the web application, the project includes a Python-based database connector for direct query execution and a Java console application that allows users to log in and retrieve data through a menu-driven interface.

# 

# The project is developed specifically for this course and is not based on previously submitted work. The GitHub repository is used to track progress, with commits showing the development process over time.

# 

# The purpose of this project is to demonstrate understanding of database design, data normalization, and integration between multiple programming languages and a relational database system.
